# Jira - importação rápida do backlog

1. Criar um projeto no Jira.
2. Importar o arquivo `jira-backlog-import.csv` ou cadastrar os itens manualmente usando o conteúdo de `product-backlog.pdf`.
3. Conferir épicos, features, histórias de usuário e enablers.
4. Criar a Sprint 1 com os itens descritos em `sprint-backlog.pdf`.
5. Exportar o backlog do produto como `product-backlog.pdf`.
6. Exportar o backlog da sprint como `sprint-backlog.pdf`.

Os PDFs já estão prontos e legíveis na raiz do repositório.
