# Figma - importação rápida dos wireframes

1. Criar um arquivo novo no Figma.
2. Arrastar as imagens da pasta `wireframes/` para o canvas.
3. Organizar as telas na ordem:
   - 01-login.png
   - 02-dashboard-aluno.png
   - 03-nova-tarefa.png
   - 04-detalhe-tarefa.png
   - 05-painel-professor.png
4. Nomear a página como `Wireframes - EduTask MVP`.
5. Exportar as telas como PNG, mantendo os mesmos nomes.
6. Manter as imagens finais na pasta `wireframes/` do repositório.
