# EduTask - MVP de Gestão Ágil

Este repositório contém a entrega do MVP da Sprint de Gestão Ágil de Projetos e Produtos.

## Produto

**EduTask** é uma plataforma de organização e acompanhamento de entregas acadêmicas. O MVP permite que alunos visualizem, cadastrem e atualizem tarefas, enquanto professores e coordenação acompanham pendências de forma objetiva.
